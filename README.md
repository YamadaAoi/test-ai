# test-ai - 测试纯 ai 编写简单前端工程能力

## 项目概况

- `vue3 SPA ts`项目，路由采用`vue-router`，状态管理采用`pinia`，UI组件库采用`elemnt-plus`

## 工程结构

```
src/
├── main.ts               # 入口
├── App.vue               # 根组件
├── stores/               # 状态管理文件夹
├── router/               # 路由文件夹
│   └── index.ts          # 路由入口文件
└── utils/                # 公共方法
└── api/                  # 请求接口
└── assets/               # 静态资源
└── hooks/                # vue hooks
└── views/                # 各路由页面
```
